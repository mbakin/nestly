# Nestly

The aim of the project is to develop a URL shortening application using NestJs, MongoDB, and Docker.


## Run Locally

Clone the project

```bash
  git clone https://github.com/mbakin/nestly.git
```

Go to the project directory

```bash
  cd nestly
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start:dev
```
